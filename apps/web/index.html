Save your update token. You'll need it to update or delete this page. Pass it back via --token in the CLI, or include it in the update_token field in a new API call to update the same URL.

Three ways to use it

CLI

npx aired file.html npx aired file.html --ttl 1h --pin 1234

Reads from stdin too: cat page.html | npx aired

MCP (Claude & other AI tools)

claude mcp add aired npx aired --mcp

Gives Claude a publish_html tool. Say "publish this as HTML" and get a live URL.

API

POST /api/publish

Send { html, ttl?, pin?, permanent? }. Get back { id, url, update_token, expiresAt }.
